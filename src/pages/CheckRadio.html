Checkbox & Radio

1. Checkboxes พื้นฐาน

ใช้สำหรับเลือกได้หลายตัวเลือกพร้อมกัน

โค้ดตัวอย่าง:
<div class="form-check">
  <input class="form-check-input" type="checkbox">
  <label class="form-check-label">
    Checkbox Default
  </label>
</div>
<div class="form-check">
  <input class="form-check-input" type="checkbox">
  <label class="form-check-label">
    Checkbox Default 2
  </label>
</div>

2. Checkboxes แบบ Disabled

เพิ่ม attribute disabled เพื่อปิดการใช้งาน checkbox

โค้ดตัวอย่าง:
<div class="form-check">
  <input class="form-check-input" type="checkbox" disabled>
  <label class="form-check-label">
    Checkbox ที่ถูกปิดใช้งาน
  </label>
</div>

3 Radio Buttons พื้นฐาน

ใช้สำหรับเลือกตัวเลือกเดียวจากหลายตัวเลือก (ต้องใช้ name เดียวกัน)

โค้ดตัวอย่าง:
<div class="form-check">
  <input class="form-check-input" type="radio" name="flexRadioDefault">
  <label class="form-check-label">
    Radio Default
  </label>
</div>
<div class="form-check">
  <input class="form-check-input" type="radio" name="flexRadioDefault">
  <label class="form-check-label">
    Radio Default 2
  </label>
</div>

4 Switches

สวิตช์เปิด/ปิดที่ใช้ markup ของ checkbox แต่เพิ่มคลาส .form-switch

โค้ดตัวอย่าง:
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" role="switch">
  <label class="form-check-label">Switch Default</label>
</div>

5 การจัดเรียงแบบ Inline

เพิ่มคลาส .form-check-inline เพื่อจัดเรียงในแนวนอน

โค้ดตัวอย่าง:
<div class="form-check form-check-inline">
  <input class="form-check-input" type="checkbox">
  <label class="form-check-label">1</label>
</div>
<div class="form-check form-check-inline">
  <input class="form-check-input" type="checkbox">
  <label class="form-check-label">2</label>
</div>
<div class="form-check form-check-inline">
  <input class="form-check-input" type="checkbox" disabled>
  <label class="form-check-label">3</label>
</div>

6. การ Binding Value กับ Model ใน Aurelia

ในกรณีที่เป็น Checkbox สามารถทำได้โดยการเพิ่ม property checked.bind
ส่วนกรณี Radio สามารถทำได้โดยการเพิ่ม attribute value.bind

checkbox1: ${checkbox1}
checkbox2: ${checkbox2}
เพศ: ${gender}
โค้ดตัวอย่าง:
<div class="form-check form-check-inline">
	<input
		class="form-check-input"
		type="checkbox"
		value="option1"
		checked.bind="checkbox1"
	/>
	<label class="form-check-label" for="inlineCheckbox1">1</label>
</div>
<div class="form-check form-check-inline mb-2">
	<input
		class="form-check-input"
		type="checkbox"
		value="option2"
		checked.bind="checkbox2"
	/>
	<label class="form-check-label" for="inlineCheckbox2">2</label>
</div>

<div class="form-check mt-3">
	<input
		class="form-check-input"
		type="radio"
		name="genderGroup"
		checked.bind="gender"
		value="male"
	/>
	<label class="form-check-label"> Male </label>
</div>
<div class="form-check mb-2">
	<input
		class="form-check-input"
		type="radio"
		name="genderGroup"
		checked.bind="gender"
		value="female"
	/>
	<label class="form-check-label"> Female </label>
</div>
