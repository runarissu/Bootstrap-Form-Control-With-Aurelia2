Form Control

1. ตัวอย่างพื้นฐาน

โค้ดตัวอย่าง:
<div>
  <label class="form-label">ชื่อ - นามสกุล</label>
  <input type="text" class="form-control" placeholder="ชื่อ - นามสกุล">
</div>

<div class="row g-3 align-items-center">
  <div class="col-auto">
    <label class="col-form-label">ชื่อ - นามสกุล</label>
  </div>
  <div class="col">
    <input type="text" class="form-control" />
  </div>
</div>

2. ตัวอย่าง Form Control ต่างๆ

input type="text" สำหรับข้อความทั่วไป
input type="number" สำหรับตัวเลข มีปุ่มเพิ่ม/ลดค่า
input type="date" สำหรับเลือกวันที่
input type="time" สำหรับเลือกเวลา
input type="tel" สำหรับหมายเลขโทรศัพท์ (รูปแบบ: 099-999-9999)
input type="url" สำหรับที่อยู่เว็บไซต์
input type="week" สำหรับเลือกสัปดาห์ในปี
input type="month" สำหรับเลือกเดือนและปี
input type="datetime-local" สำหรับเลือกวันที่และเวลา
input type="file" สำหรับอัปโหลดไฟล์
input type="file" multiple สำหรับอัปโหลดหลายไฟล์

3. การปิดใช้งาน (Disabled)

ปิดใช้งาน Form Control ได้โดยเพิ่มแอตทริบิวต์ disabled และ readonly

โค้ดตัวอย่าง:
<input class="form-control" type="text" value="Input Field with Readonly" disabled readonly>

4. การ Binding Value กับ Model ใน Aurelia

Binding Value ได้โดยเพิ่มแอตทริบิวต์ value.bind

First Name: ${firstName}
โค้ดตัวอย่าง:
<input class="form-control" type="text" value.bind="firstName">
