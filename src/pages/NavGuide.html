Nav

1. Base Nav

การนำทางพื้นฐานใน Bootstrap ใช้คลาส .nav ซึ่งสร้างด้วย flexbox

  • Active
  • Link
  • Link
  • Disabled
โค้ดตัวอย่าง:
<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled">Disabled</a>
  </li>
</ul>

2. Horizontal Alignment

สามารถเปลี่ยนการจัดวางแนวนอนของ nav ด้วย flexbox utilities ได้ โดยค่าเริ่มต้น nav จะจัดชิดซ้าย แต่สามารถเปลี่ยนเป็นจัดกึ่งกลางหรือชิดขวาได้

2.1 จัดกึ่งกลางด้วย .justify-content-center
  • Active
  • Link
  • Link
  • Disabled
โค้ดตัวอย่าง:
<ul class="nav justify-content-center">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled">Disabled</a>
  </li>
</ul>
2.2 จัดชิดขวาด้วย .justify-content-end
  • Active
  • Link
  • Link
  • Disabled
โค้ดตัวอย่าง:
<ul class="nav justify-content-end">
  <li class="nav-item">
    <a class="nav-link active"href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" >Disabled</a>
  </li>
</ul>

3. Vertical

สามารถจัดเรียง navigation ในแนวตั้งได้โดยเปลี่ยนทิศทางของ flex item ด้วย utility .flex-column หากต้องการจัดเรียงในแนวตั้งเฉพาะบางขนาดหน้าจอ สามารถใช้เวอร์ชัน responsive ได้ (เช่น .flex-sm-column)

  • Active
  • Link
  • Link
  • Disabled
โค้ดตัวอย่าง:
<ul class="nav flex-column">
  <li class="nav-item">
    <a class="nav-link active" href="#">Active</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled">Disabled</a>
  </li>
</ul>
