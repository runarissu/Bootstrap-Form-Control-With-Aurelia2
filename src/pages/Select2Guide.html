Select2

1. Single Select

โค้ดตัวอย่าง:
<select2
	items.bind="ProvincesList"
	calculate-item-id="code"
	calculate-item-text="name"
	placeholder="Please select .."
	is-allowed-clear="true">
</select2>

โดยสามารถเพิ่ม property enable-search-box="true" เพื่อให้สามารถพิมเพื่อค้นหาได้

2. Multiple Select

โค้ดตัวอย่าง:
<select2
	items.bind="ProvincesList"
	calculate-item-id="code"
	calculate-item-text="name"
	is-multiple="true"
	placeholder="Please select .."
	is-allowed-clear="true">
</select2>

3. การปิดใช้งาน Select2 (Disable)

เพิ่ม property disabled.bind="true" เพื่อปิดการใช้งาน select2

โค้ดตัวอย่าง:
<select2
	items.bind="ProvincesList"
	calculate-item-id="code"
	calculate-item-text="name"
	placeholder="Please select .."
	is-allowed-clear="true"
	disabled.bind="true">
</select2>

4. การ Binding Value กับ Model ใน Aurelia

Binding Value ได้โดยเพิ่ม property selected-values.bind

จังหวัด: ${SelectedProvince}
โค้ดตัวอย่าง:
<select2
	items.bind="ProvincesList"
	calculate-item-id="name"
	calculate-item-text="name"
	placeholder="Please select .."
	is-allowed-clear="true"
	selected-values.bind="SelectedProvince">
</select2>

กรณีที่ต้องการ Binding Object ทำได้โดยเพิ่ม property selected-items.bind

รหัสจังหวัด: ${SelectedProvinceItem.code}
จังหวัด: ${SelectedProvinceItem.name}
โค้ดตัวอย่าง:
<select2
	items.bind="ProvincesList"
	calculate-item-id="code"
	calculate-item-text="name"
	placeholder="Please select .."
	is-allowed-clear="true"
	selected-items.bind="SelectedProvinceItem">
</select2>
