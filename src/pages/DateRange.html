Date Range

1. พื้นฐาน

สามารถเลือกวันที่เริ่มต้นและวันที่สิ้นสุดได้

โค้ดตัวอย่าง:
<date-range-picker></date-range-picker>

2. กำหนด Min Date และ Max Date ด้วย Aurelia

สามารถตั้งค่า Min Date และ Max Date โดยการเพิ่ม property min-date.bind และ max-date.bind

สามารถกำหนดได้โดยใช้ฟอร์แมตดังนี้

  • แบบ String เช่น 
    today = "19/03/2025";
maxDate = "25/03/2025";
  • แบบ DateObject เช่น 
    today = new Date();
maxDate = today.getDate() + 14
โค้ดตัวอย่าง:
<date-range-picker
	min-date.bind="today"
	max-date.bind="maxDate"
></date-range-picker>

3. การ Binding Value กับ Model ใน Aurelia

สามารถ Binding Value ได้โดยการเพิ่ม property start-date.bind และ end-date.bind

วันที่เข้าพัก: ${startDate}
วันที่เช็คเอ้าท์: ${endDate}
โค้ดตัวอย่าง:
<date-range-picker
	start-date.bind="startDate"
	end-date.bind="endDate"
></date-range-picker>
